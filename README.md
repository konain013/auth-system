# Authentication System

A secure and scalable user authentication system built using the MERN backend stack (Node.js, Express, MongoDB, Mongoose). This project implements core authentication features including user registration, login, password hashing, and JWT-based session management, following a clean MVC (Model-View-Controller) folder structure for maintainability and scalability.

## Features
- User registration and login
- Secure password hashing (bcrypt)
- JWT-based authentication
- Environment-based configuration (dotenv)
- Modular structure (config, controllers, models, routes, middleware)

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose ODM
- **Auth:** JSON Web Tokens (JWT), bcrypt
